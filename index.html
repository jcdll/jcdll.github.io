# 📚Blog

这是一个兼具博客文章、知识管理、文档查找的个人网站，主要内容是 Web 前端技术。如果你喜欢这个博客&主题欢迎到GitHub点个 Star、获取源码，或者交换友链 ( •̀ ω •́ )✧

# 🎨Theme

本站主题是根据VuePress的默认主题修改而成。取名Vdoing(维度)，含义是方便你多维度的快速的管理和查找学习笔记以及技术文档。添加方便管理笔记文档的目录页、自动生成侧边栏工具、搜索框功能扩展等，添加博客所需的评论、时间轴、分类、最近更新等。更多详情。
